# AI SOC Assistant

## Overview

AI SOC Assistant is a cybersecurity automation project built in Ubuntu Linux using Python.

The project simulates Security Operations Center (SOC) workflows by analyzing Linux authentication logs and generating automated security findings and investigation recommendations.

This project focuses on:
- Security automation
- Authentication log analysis
- SOC investigation workflows
- Linux administration
- Python scripting
- AI-driven cybersecurity concepts

---

## Features

Current functionality includes:
- Reading Linux authentication logs
- Detecting failed password attempts
- Detecting authentication failures
- Detecting invalid usernames
- Detecting sudo/admin activity
- Risk scoring logic
- Severity level classification
- Automated investigation recommendations

## Detection Logic

The script currently analyzes logs for:
- Failed password attempts
- Authentication failures
- Invalid usernames
- Sudo activity

The script then:
- Counts suspicious events
- Calculates a risk score
- Assigns a severity level
- Generates investigation guidance
